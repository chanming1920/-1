# 프로젝트 구상
## 패스워드 분석
기본 패스워드 정책: 
1. 알파벳(대,소) + 숫자 + 특수문자를 2가지 종류 이상 사용하여 조합한다.
2. 2가지 조합인 경우 10자리 이상을 권장한다.
3. 3가지 이상의 조합인 경우 8자리 이상을 권장한다.
4. 같은 문자가 3번 연속 사용되는 것을 권장하지 않는다 / 연속적인 숫자와 반복 사용을 권장하지 않는다.
## 하위기능 추출
1. 패스워드 입력
2. 패스워드 정보받기
3. 패스워드 정보 변환
4. 변환된 서치기능 통해 패스워드 구성 확인
5. 정책에 따른 판독
6. 판독에 따른 유저에게 알림
## 목표
1차 패스워드 입력 후 정책에 따른 기능 추상화
